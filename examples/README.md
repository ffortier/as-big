# Examples for `as-big` 

A collection of examples for [`as-big`](https://github.com/ttulka/as-big).

See contennt of [`assembly`](https://github.com/ttulka/as-big/tree/main/examples/assembly) directory.

## Build

```sh
npm i
npm run asbuild:release
```

## Test

```sh
npm test
```

## Run

```sh
node index.js
```
